# Zyra – Daily Planner & Task Manager

Zyra is a lightweight Chrome extension that combines spreadsheet-style planning with smart task execution.

## Features

- Spreadsheet-style planner
- Inline editing
- Task dependencies (auto-blocking)
- Smart reminders
- Google Calendar quick-add
- Local-first (no backend)

## Installation

1. Open chrome://extensions/
2. Enable Developer Mode
3. Click "Load Unpacked"
4. Select this folder

## Usage

- Add tasks directly in the grid
- Track progress with checkboxes
- Use dependencies to avoid mistakes
- Schedule tasks using the calendar button

## Privacy

All data is stored locally in your browser.
No external servers are used.
